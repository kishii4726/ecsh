# ecsh
ecsh is a tool to execute ECS exec with ease.

# Install
```
$ wget https://github.com/kishii4726/ecsh/releases/download/v0.0.2/ecsh_v0.0.2_linux_amd64.zip

$ unzip ecsh_v0.0.2_linux_amd64.zip ecsh

$ sudo cp ecsh /usr/local/bin
```

# Usage
```
$ ecsh
```
